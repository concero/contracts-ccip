# Solidity API

## ParentPoolStorage

### s_maxDeposit

```solidity
uint256 s_maxDeposit
```

variable to store the max value that can be deposited on this pool

### s_loansInUse

```solidity
uint256 s_loansInUse
```

variable to store the amount that will be temporary used by Chainlink Functions

### s_donHostedSecretsSlotId

```solidity
uint8 s_donHostedSecretsSlotId
```

variable to store the Chainlink Function DON Slot ID

### s_donHostedSecretsVersion

```solidity
uint64 s_donHostedSecretsVersion
```

variable to store the Chainlink Function DON Secret Version

### s_getBalanceJsCodeHashSum

```solidity
bytes32 s_getBalanceJsCodeHashSum
```

### s_ethersHashSum

