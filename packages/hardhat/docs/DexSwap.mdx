## DexSwap_EmptyDexData

```solidity
error DexSwap_EmptyDexData()
```


## DexSwap_InvalidPath


## DexSwap

### DexSwap_OrchestratorContractUpdated

```solidity
event DexSwap_OrchestratorContractUpdated(address previousAddress, address orchestrator)
```

event emitted when the orchestrator address is updated

### DexSwap_RemovingDust

```solidity
event DexSwap_RemovingDust(address receiver, uint256 amount)
```

event emitted when value locked in the contract is removed

### constructor

```solidity
constructor(address _proxy, address[3] _messengers) public
```

### entrypoint

```solidity
function entrypoint(struct IDexSwap.SwapData[] _swapData, address _recipient) external payable
```

Entry point function for the Orchestrator to take loans

_only the Orchestrator contract should be able to call this function_

#### Parameters

| Name | Type | Description |
| ---- | ---- | ----------- |
| _swapData | struct IDexSwap.SwapData[] | a struct array that contains dex information. |
| _recipient | address |  |
