### setClfPremiumFees

```solidity
function setClfPremiumFees(uint64 _chainSelector, uint256 feeAmount) external payable
```

Function to update Chainlink Functions fees

_arb default fee: 3478487238524512106 == 4000000000000000; // 0.004 link
base default fee: 10344971235874465080 == 1847290640394088; // 0.0018 link
opt default fee: 5224473277236331295 == 2000000000000000; // 0.002 link_

#### Parameters

| Name | Type | Description |
| ---- | ---- | ----------- |
| _chainSelector | uint64 | Chain Identifier |
| feeAmount | uint256 | fee amount |
